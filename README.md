## Objective
Perform data cleaning and exploratory data analysis (EDA) to identify patterns and trends in the Titanic dataset.

## Dataset
Titanic dataset loaded using Seaborn library.

## Steps Performed
- Checked dataset structure and summary statistics
- Handled missing values in age, embarked, and deck columns
- Performed exploratory data analysis using bar plots, histograms, and heatmaps
- Analyzed relationships between survival, gender, class, and age

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Insights
- Females had a higher survival rate than males
- First-class passengers had higher survival chances
- Fare and passenger class were strongly related to survival
